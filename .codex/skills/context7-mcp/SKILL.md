# Context7 MCP Skill

Use this skill when the task depends on third-party documentation that is not already captured inside the repository.

## Rules

- Prefer local repo docs first.
- Use explicit, narrow queries.
- Bring back only the minimum external context needed to unblock the task.
- Do not use this skill for purely local codebase questions.
