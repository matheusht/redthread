# Research Skill

Use this skill when the goal is understanding rather than editing.

## Workflow

1. Start at `README.md` and `docs/AGENT_DECISION_TREE.md`.
2. Load only the focused docs needed for the question.
3. Use targeted search to find real source files on the execution path.
4. Summarize findings with paths, architecture notes, and constraints.
5. Call out stale docs or mismatches between guidance and the live tree.

## Output

- key paths
- current architecture summary
- repo conventions and SOPs
- risks, inconsistencies, or missing references

## Guardrails

- Do not modify files while using this skill.
- Prefer short synthesized notes over long excerpts.
